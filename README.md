An AI-powered Resume Analyzer built with **React**, **TypeScript**, **React Router v7**, and **Puter.js**. The application helps users analyze their resumes against job descriptions using AI-generated insights, ATS scoring, and personalized feedback to improve their chances of landing interviews.

## 🌐 Live Demo

🔗 **Website:** https://ai-resume-analyzer-87xw-lqqwjg.puter.site/

## ✨ Features

* 🔐 Authentication using Puter.js
* 📄 Upload and manage resumes in PDF format
* 🤖 AI-powered resume analysis
* 📊 ATS (Applicant Tracking System) compatibility scoring
* 💡 Personalized feedback and suggestions
* 🎯 Resume matching against job descriptions
* ☁️ Cloud file storage using Puter File System
* 🗄️ Key-value data storage with Puter KV Store
* 📱 Responsive and modern UI

## 🛠️ Tech Stack

### Frontend

* React
* TypeScript
* React Router v7
* Tailwind CSS

### Backend Services

* Puter.js

  * Authentication
  * File System (FS)
  * Key-Value Store (KV)
  * AI APIs

* Vite

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js (v18 or later)
* npm

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Y45hK1r0l4/AI-Resume-Analyzer.git
```

2. Navigate to the project directory:

```bash
cd AI-Resume-Analyzer
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```
